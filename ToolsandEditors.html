Web Editors

How do I even make a web page?

The answer to this question is actually a lot simpler than you think. Many operating systems come with a native text editor for web pages. For example, you can use notepad on Windows OS to make a simple HTML Web Page. Are there better ones though? Oh Yeah! Web Editors to developers are stoves to chef's, you basically can't do your job without them. There are many different kinds of web editors our there and some more popular than others. To name a few are Notepad++, Sublime, and WebStorm. While they all can edit a web page, or web app, they all offer different amenitites and powerful tools. Some you have to pay just to use them! As a web developer, it is good to know how to use more than one editor. Why? Well depending on what company you work for or if you're working from home, the team you work with may not be familiar with the editor you use. Or the company will flat out not pay for something like WebStorm that has a subscription fee. Either way, web editors are a necessary skill for any developer.

Google Dev Tools

What tools should I have?

Google Dev Tools is a great resource for every developer to know. It's availabe both on Google Chrome and Mozilla Firefox browsers. It allows you as a developer to analyze code from the user's point of view. If a page is not rendering correctly on a page, using the GDT function will bring up a window to help you locate what the issue is. Also, it allows you to see how a page or app renders on a mobile device. This is crucial part of web development as more and more sites are aiming for mobile friendliness.

Toolkits

What is a toolkit?

Another important aspect of web development is having at least some basic knowledge of a web toolkit. We have learned about Editors and Google Dev Tools, but another resource of value is Web Toolkits. Anything you know from languages, frameworks, fullstacks, editors and other software are part of what's called your toolkit. You can maintain and edit websites from the ground up or you can use frameworks. A Web Toolkit however is a set of tools that helps you maintain, update, or restructure websites. Google has provided many resources such as the Google Web Tookit, Polymer and app Broilerplates to name a few tools. They assist in a number of different coding aspects that could speed up the process of development.
