Server-Side Scripting

Languages

Common languages for Server-Side Scripting are PHP, Java, Perl, Python, Ruby, etc. Processing is done on the server. And then you get a new processed page when you refresh it on your browser. Personal Home Page (PHP) is the most common and widely used of all Server-Side Languages. It is easily paired with most toolkits and is great when run on a Linux OS. Most developers agree it is an easy language to learn and transfers understanding to other langauges such as Python or Ruby.

Databases

What is a Database?

A database is like a vault or collection of data that stores information on a server. The database can add, delete or manipulate the data in order to extrapolate certain information. Many Back-End Developers use Structured Query Language (SQL) as their primary means of manipulating databases. It is the most widely used language for databases by developers. However, it is important to know that certain languages only work on certain databases. For example, SQL only works on SQL databases that are on Microsoft Servers. So as a developer, make sure you are using the right languages for the right database.

Server-Side Applications

Framework Languages

Depending on the needs of the website, each Back-End developer needs to know which frameworks for their applications to use. Frameworks just like Front-End frameworks, are pre-existing codes that help speed up development. The pros of choosing a framework for a specific language is that it optimizes a langauges use for that particular project. For example, Ruby on Rails allows the "gems" (monoliths of pre-existing code) to be placed in the server faster. The cons of frameworks are just that of Front-End. You are limited to what you can do with the code you are developing. Also, not all Frameworks support other languages. While ASP.NET from Microsoft supports multiple languages, Django only supports Python.

Full Stack

The Whole Enchilada

Full Stack developers are deservingly called so because they have a full toolkit. They are able to manipulate all three areas of back-end and usually front-end as well. Two popular full stack toolkits are in demand right now. LAMP (Linux/Apache/MySQL/PHP, Perl or Python) and MEAN (MongoDB, Express.js, Angular.js, Node.js). Lamp is typically the most widely known of the two. However, Mean has gained headway and some developers consider it the future of FullStack as web technology is headed for more web Apps and more cloud services.
