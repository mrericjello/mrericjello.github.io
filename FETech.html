Basic Languages to Know

HTML5 Hyper Text Markup Language

HTML is the basic most coding language every Front-End developer must learns starting out. It is essentially structuring a document like in MS Word, but on the internet. You can add titles to the pages, headings, lists, body, text, etc. The current most version of HTML is HTML 5. The W3C made a standardiztion of HTML as the bases for all browsers to render web pages. Every webpage is made from HTML.

CSS3 Cascading Style Sheets

CSS is a Front-End language that styles certain portions of the HTML document. It specifies specific elements in the HTML code and tells how to specifically style the selected area with properties and values. It allows for modular site designs.

.JS Java Script

Javascript is a scripting language that was developed in 1995 by Netscape to increase functionality on webpages. It allows the addition of interactive elements, updating data in the browser.
