Web Sites

Web Pages

Originally the web started out as simply web pages. Pages that displayed only information for users to read and that was all. In fact many website such as Wikipedia still use the simple web page design with minimal function for users to easily find what they are looking for. As a web developer, it is important to know if making a solution for clients requires only a simple web page. That way, it makes your job easier with less code to write and manipulate.

Web Apps

Web Applications

The web is continually evolving and heading in new directions. One distinct trend is the increase of web applications being developed. A web application differs from that of a web page in that it performs a specific function to create, edit, or manipulate data. For example, Pixlr Editor is an online graphic design editor that edits images. Many web sites have begun to push not only for mobile use, but also to transform their websites into mobile apps so that users can still access the information on the website without having the need for the full site itself. There are drawbacks in which not every mobile app has a computer site, and not ever website has a mobile app. Bridging the gap between these two is simply up to the developers, and can be exclusive one way or another. Mobile Applications are great because they load quickly on mobile devices, and mimic the native mobile os apps already installed. So no need to fiddle around figuring out how to use a mobile app. Also, apps data-bind which mean they are responsive to touch, entering information, etc.
